## Theory

### 1. Python Sets

A **Set** is a collection which is both unordered and unindexed. In Python, sets are written with curly brackets `{}`. 

* 
**Unordered:** The items in a set do not have a defined order and cannot be referred to by an index. 


* 
**Unique Elements:** Sets do not allow duplicate values; if a duplicate is added, it is ignored. 


* 
**Dynamic Membership:** While the items are unchangeable, you can add new items or remove existing ones at any time. 



### 2. Python Dictionaries

A **Dictionary** is a collection used to store data values in `key:value` pairs. It is similar to a real-world map or directory. 

* 
**Keys and Values:** Each element is a pair consisting of a unique key and an associated value. 


* 
**Identifiers as Keys:** Keys must follow the rules for identifiers (case-sensitive, no starting digits) and must be unique. 


* 
**Mutable:** Dictionaries allow for the changing, adding, or removing of items after the dictionary has been created. 



### 3. Basic Syntax and Functions

Python provides specific built-in functions to interact with these data types: 

* 
**`type()`**: Used to identify if a variable is a `set` or a `dict`. 


* 
**`len()`**: Returns the number of items in the collection. 


* 
**`set()` / `dict()**`: Constructor functions used to convert other data types (like lists) into sets or dictionaries. 


* 
**Input/Output**: Values can be accepted via `input()` and displayed using `print()`. 



### 4. Operators for Collections

* 
**Membership Operators (`in`, `not in`)**: Used to check if a specific key or element exists within the collection. 


* 
**Set Operators**: Includes Union (`|`), Intersection (`&`), and Difference (`-`) to compare multiple sets. 


* 
**Assignment Operators**: Used to update dictionary values or add new elements (`=`, `+=`).
